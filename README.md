Automatic Night Light System

This project is an automatic night light circuit designed in Proteus using an LDR sensor and a BC547 transistor. The system automatically controls the LED depending on the ambient light level.

When the environment becomes dark, the resistance of the LDR changes and activates the BC547 transistor, allowing the LED to turn on automatically. When light is detected, the LED turns off.

## Components Used

BC547 Transistor

LDR (Light Dependent Resistor)

LED

4700 Resistor

120kQ Resistor
